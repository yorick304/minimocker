# minimocker

> A mini mocker your local development

## quick start

```javascript
sudo npm i -g minimocker
mock
```

## config

```bash
Options:

    -h, --help                   output usage information
    -u, --uiport <uiport>        set port of control panel
    -p, --proxyport <proxyport>  set port of proxy server
```
